예제 코드 실행 방법
-----------------------

각 파일은 세 부분으로 구성되며, 버전별로 01, 02, ... 식으로 일련 번호가 붙어있다.

- Aop_nn.js는 테스트 대상 코드다.
- 테스트 코드는 Aop_nn_tests.js에 있다.
- index_nn.js는 재스민으로 테스트를 실행하기 위한 HTML 파일이다.

최종본은 끝부분에 'nn'이 없다. Aop.js을 테스트하려면 index.html에서 Aop_tests.js를 실행한다.

세 부분을 실행하려면 웹 브라우저에서 index_nn.html 파일을 그냥 열면 된다.
